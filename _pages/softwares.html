MIAMI

For studying marker co-expression in multiplex imagingdata.

MIAMI

This R package is associated with the paper: "MIAMI: Mutual Information-based Association Analysis of Multiplex Imaging data." Available here.

RCFGL

For estimating multiple Gaussian Graphical networks.

RCFGL

This C and Python based package is associated with the manuscript: "Rapid condition-adaptive Fused Graphical Lasso". Available here.

DenVar

For studying marker influence on survival in mIHC and MIBI data.

DenVar

This R package is associated with the paper: "DenVar: Density based Variation analysis of multiplex imaging data." Available here.

PredLMM

For estimating SNP heritability.

PredLMM

This Python based package is associated with the paper: "Rapid Estimation of SNP Heritability using Predictive Process approximation in Large scale Cohort Studies". Available here.

VectraMIBI

For cell-phenotyping in multiplex imaging data.

VectraMIBI

This R package is associated with the paper "On Clustering for Cell Phenotyping in Multiplex Immunohistochemistry (mIHC) and Multiplexed Ion Beam Imaging (MIBI) Data". Available here.

RMultiPAR

For genetic association analysis.

RMultiPAR

This R based package is associated with the paper "Multivariate Association Analysis of Correlated Traits in Related Individuals." Available here.
